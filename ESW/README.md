## ESW predictions

The ESW is predicted via predictions of the reduction and oxidation potentials. The predictions have been performed using the regression method. The training dataset is adapted from Wang et al. and contains 1502 materials from the ICSD.


(1) Wang, X.; He, B.; Liu, B.; Avdeev, M.; Shi, S. A Database of Electrochemical Stability Windows Containing over 1500 Solid‐State Inorganic Compounds. Adv Funct Materials 2024, 2406146.

(2) Zagorac, D.; Müller, H.; Ruehl, S.; Zagorac, J.; Rehme, S. Recent Developments in the Inorganic Crystal Structure Database: Theoretical Crystal Structure Data and Related Features. Journal of Applied Crystallography 2019, 52 (5), 918–925. https://doi.org/10.1107/s160057671900997x.
